Создаём базовый шаблон HTML 5


Чтобы обозначить тот или иной тег на странице, к демонстрации мы подключим подсветку тегов. 
Для этого используем служебный стилевой файл outline.css. 

В шапке страницы появился ещё один служебный тег — <base>, 
с помощью него мы сократим путь до используемых ресурсов. 

В качестве значения атрибута href укажем базовый адрес (URL) 
полного пути, по которому расположены подключаемые ресурсы. 

Подробнее о теге <base> можно прочитать в спецификации.

В блоке main выделяются три раздела: главная новость, обзоры и промо-блок. Эти разделы неразрывно связаны с тематикой сайта, 
поэтому размечаем их элементами <section>. Добавим атрибут class для разделов со значениями: headline — главная новость, 
сенсация, reviews — обзоры и promo — промо-блок. Подвал в данном случае содержит простые элементы, и крупные блоки для разметки 
не понадобятся.

Приступаем к добавлению заголовков для всей страницы и для смысловых разделов, выделенных на предыдущем шаге. Макет не содержит 
заголовок, который подошёл бы для всей страницы. Добавим его. Поскольку это главная страница сайта о культурных событиях, для неё 
вполне подойдёт заголовок «Новости культуры». Так мы чётко обозначили тематику сайта. Этот заголовок отсутствует на макете, добавим 
ему атрибут class="visually-hidden" для того, чтобы доступно его скрыть. Подробно узнать о том, как правильно прятать, можно 
в HTML-шорте «Как прятать». У блока с главной новостью есть свой заголовок — «Подводный отель», используем его. Это будет заголовок 
второго уровня. Раздел с обзорами и раздел промо-блока имеют свои заголовки на макете, добавим их в разметку.
